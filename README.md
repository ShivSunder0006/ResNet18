# ResNet-18 Interactive Explainer
An interactive Streamlit application designed to demystify the [Deep Residual Learning for Image Recognition](https://arxiv.org/abs/1512.03385) paper. Includes dynamic RAG interaction, live feature mapping, and flowchart models.
